   ```bash
   $ cd BackCheck
   ```

4. Install the requirements:

   ```bash

   ```

5. Make a copy of the example environment variables file:

   ```bash
   $ cp env_example .env
   ```

6. Fill the environment variables, contained in the .env file.

7. Run the backend app:

   ```bash
   $ python3 app.py
   ```
